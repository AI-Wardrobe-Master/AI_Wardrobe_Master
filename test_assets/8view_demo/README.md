# 8-View 3D Demo Assets

This folder keeps the original demo outputs imported into the local database for presentation and testing.

Imported demo database records:

- Card pack: `8d000000-0000-4000-8000-00000000c001`
- Public wardrobe: `8d000000-0000-4000-8000-00000000f001`
- Clothing items:
  - `8d000001-0000-4000-8000-000000000001` - Blue blazer 8-view demo
  - `8d000002-0000-4000-8000-000000000002` - Camel belted coat 8-view demo
  - `8d000003-0000-4000-8000-000000000003` - Floral long dress 8-view demo

Each numbered folder contains the source image, background-removed image, 8 rendered angle images, and GLB meshes. The application serves the imported files from backend CAS blob storage, not directly from this folder.
