# ai_wardrobe_app

A new Flutter project for **AI Wardrobe Master** — 智能衣柜与穿搭资产管理。

## 项目结构

```
├── android/     # Android 平台
├── ios/         # iOS 平台
├── lib/         # Dart 应用代码
├── linux/       # Linux 桌面
├── macos/       # macOS 桌面
├── web/         # Web
├── windows/     # Windows 桌面
├── test/        # 测试
├── pubspec.yaml # 依赖配置
└── README.md
```

## 环境要求

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Dart 3.x

## 本地运行

```bash
# 安装依赖
flutter pub get

# 选择设备后运行（如 Chrome）
flutter run -d chrome
```
