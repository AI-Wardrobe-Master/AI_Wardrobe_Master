# AI Wardrobe Master

AI Wardrobe Master 团队协作仓库：Flutter 应用与组员文档。

## 目录结构

```
ai_wardrobe/
├── ai_wardrobe_app/          # Flutter 应用（智能衣柜）
│   ├── lib/
│   ├── android/, ios/, web/ ...
│   └── README.md
├── groupmembers'markdown/     # 组员 markdown（进度、分工等）
└── README.md
```

## 快速开始

```bash
# 运行 Flutter 应用
cd ai_wardrobe_app
flutter pub get
flutter run -d chrome
```

## 仓库说明

- **ai_wardrobe_app**：主应用代码，详见 [ai_wardrobe_app/README.md](ai_wardrobe_app/README.md)
- **groupmembers'markdown**：组员文档与分工记录
